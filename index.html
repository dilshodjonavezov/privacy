Политика конфиденциальности

Последнее обновление: 14 апреля 2026

Настоящая Политика конфиденциальности описывает, как приложение «Касбнамо» собирает, использует и защищает информацию пользователей.

1. Какие данные могут использоваться

Приложение может использовать следующие данные в рамках своей работы:

  • информация, необходимая для работы функций приложения;
  • результаты тестирования и прогресс пользователя;
  • локально сохраненные данные и настройки;
  • изображения, выбранные пользователем, если используется функция выбора фото.

2. Как используются данные

Данные используются только для следующих целей:

  • предоставление основных функций приложения;
  • сохранение пользовательских настроек и результатов;
  • улучшение удобства использования приложения;
  • обеспечение корректной работы отдельных функций.

3. Хранение данных

Часть данных может храниться локально на устройстве пользователя. Если приложение использует сетевые функции, передача данных выполняется с использованием стандартных механизмов безопасности платформы.

4. Передача данных третьим лицам

Приложение не продает персональные данные пользователей и не передает их третьим лицам, за исключением случаев, когда это требуется законом или необходимо для работы используемых системных сервисов.

5. Права пользователя

Пользователь может:

  • удалить приложение со своего устройства;
  • очистить локальные данные приложения, если это поддерживается устройством или системой;
  • обратиться к разработчику по вопросам конфиденциальности.

6. Конфиденциальность детей

Если приложение используется несовершеннолетними, обработка данных должна происходить с соблюдением применимых требований законодательства.

7. Изменения политики

Настоящая Политика конфиденциальности может обновляться. Актуальная версия публикуется на этой странице.

8. Контакты

По вопросам, связанным с конфиденциальностью, можно связаться с разработчиком: dmgroup.app.support@gmail.com
